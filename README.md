# EpicOS
EpicOS is an interactive webOS where you can write notes, explore our solar system, use a calculator, play a little racing game, and much more! EpicOS is a fun and helpful OS for your needs.

# Features:
+ Closeable/Openable and Draggable windows
+ Live clock that shows the date and time
+ A note-taking application that can save and load your most recent notebook (if desired)
  + Has up to two saves!
+ A live space application which shows you the current position of the planets in our solar system
  + You can move around and zoom in/out throughout the Solar System
+ A helpful calculator that performs basic mathematical functions.
  +The calculator also has a conversion mode which can convert from several common units such as: Mi-Km and °F-°C (and vice versa)
+ A racing game where you race around a track and set the fastest time you can!
+ A new weather app where you simply type in your city and can view the weather!
  + Includes helpful data such as the high, low, wind speed, wind direction, sunrise/sunset, precipitation, and a 7-day forecast.
+ You can now customize your wallpaper!
